Integração com N8N

Não configurado
Como funciona
Toda mensagem recebida no WhatsApp é encaminhada para o seu N8N via webhook. O N8N processa a mensagem com a lógica que você montar e chama o VETRA para responder.
WhatsApp → VETRA recebe → N8N → VETRA /send → WhatsApp
1 URL do Webhook no N8N
Cole aqui a URL do nó Webhook que você criar no N8N. No N8N: clique em + → Trigger → Webhook → copie a URL de teste.
2 Payload que o N8N recebe 
{
  "event": "message_received",
  "phone": "5515999991234",
  "name": "João Silva",
  "content": "Quero simular meu consignado",
  "type": "text",
  "timestamp": "2026-05-17T15:30:00.000Z",
  "account_id": "uuid-da-sua-conta",
  "media_id": null,
  "media_mime_type": null
}
No N8N: use {{ $json.phone }}, {{ $json.name }}, {{ $json.content }} etc. nos nós seguintes.
3 Como enviar resposta pelo N8N
Adicione um nó HTTP Request no N8N com estas configurações:
Método: POST
URL:
Body (JSON):
{
  "to": "{{ $json.phone }}",
  "message": "Olá, {{ $json.name }}! Aqui está sua resposta...",
  "account_id": "{{ $json.account_id }}"
}
💡 Dica importante: No N8N, durante desenvolvimento use a URL de teste do Webhook. Quando quiser que funcione 24h, clique em Activate no N8N e use a URL de produção (troca /webhook-test/ por /webhook/).

☁️ Onde rodar o N8N: Use n8n.io Cloud (plano gratuito disponível) ou instale no seu servidor.
